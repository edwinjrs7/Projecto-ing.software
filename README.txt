INSTRUCCIONES DE EJECUCIÓN
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Extraemos el archivo ZIP y pegamos la carpeta del proyecto en la carpeta htdocs de XAMPP.
2. Luego en nuestro navegador de preferencia en la barra de direcciones pegamos la siguiente dirección: http://localhost/proyecto_software/index/
3. Disfrutamos de nuestra aplicación web.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

INSTRUCCIONES PARA EJECUTAR LA BASE DE DATOS

IMPORTANTE: el proyecto está dividido o seccionado mediante carpetas por lo que el "SCRIPT SQL" de la base de datos se ubica en la misma carpeta donde se encuentra el archivo de conexión a la base de datos.

Nuestro proyecto usa como motor de base de datos MySQL por lo que se puede ejecutar mediante MySQL Workbech o en PHPMyAdmin.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

OPCION 1: 

1. Abrimos nuestro editor de base de datos (MySQL Workbech).
2. Cuando ya estemos ubicados en nuestro gestor de Bases de Datos nos dirigimos al menú de arriba y escogemos la opción "File".
3. Buscamos la opción llamada "Open SQL Script" y le damos click 
4. Buscamos la dirección de nuestro proyecto en este caso es C:\xampp\htdocs\Proyecto_software
5. ingresamos a la carpeta "index" y luego a la carpeta "bdd"  y seleccionamos en el archivo "Script_proyecto_sql.sql"
6. Damos click en abrir.

OPCION 2:

1. Abrimos nuestro Explorador de Archivos.
2. En la barra de  direcciones pegamos la ubicación del proyecto con el archivo : C:\xampp\htdocs\Proyecto_software\index\bdd
3. Damos click y abrimos el archivo "Script_proyecto_sql.sql" en MySQL Workbech
